  <action>
    In each of the three files add `from fastapi.concurrency import run_in_threadpool` and wrap EVERY synchronous Supabase `.execute()` call that runs inside an `async def` so it is offloaded to a threadpool. Pattern:

    Before:
    ```python
    result = db.table("leads").select("id, notes").eq("ghl_contact_id", contact_id).execute()
    ```
    After:
    ```python
    result = await run_in_threadpool(
        lambda: db.table("leads").select("id, notes").eq("ghl_contact_id", contact_id).execute()
    )
    ```
    Apply to:
    - webhooks.py: the select+update in handle_inbound_message (leads select, leads notes update, sms_sequences update) and handle_contact_tag_added (leads select, leads status update).
    - callrail_webhooks.py: the clients select, calls existing select, calls insert, and the leads insert (keep the leads insert inside its existing try/except for the unique-violation dedup — wrap the `.execute()` call, not the try/except structure).
    - stripe_service.py: every `db.table(...).().execute()` in handle_subscription_created, handle_subscription_deleted, handle_invoice_paid (clients select + billing insert), handle_invoice_failed.
    Do not change any query logic, column lists, or control flow — only wrap the execute() calls. Do not switch to an async client.
  </action>

  <files>api/config.py, api/main.py</files>

  <action>
    Create api/config.py:
    ```python
    import os

    REQUIRED_ENV_VARS = [
        "SUPABASE_URL",
        "SUPABASE_SERVICE_ROLE_KEY",
        "STRIPE_WEBHOOK_SECRET",
        "CALLRAIL_WEBHOOK_SECRET",
    ]


    def validate_required_env() -> None:
        """Fail fast at boot if any required env var is missing or empty."""
        missing = [name for name in REQUIRED_ENV_VARS if not os.environ.get(name)]
        if missing:
            raise RuntimeError(
                "Missing required environment variables: "
                + ", ".join(missing)
                + ". Set them in Railway (or your local .env) before starting the service."
            )
    ```
    In api/main.py: import validate_required_env from api.config and call it as the FIRST statement inside lifespan(), before init_supabase():
    ```python
    from api.config import validate_required_env
    ...
    async def lifespan(app: FastAPI):
        validate_required_env()
        init_supabase()
        yield
    ```
